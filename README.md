## Features
* ESP32-S2FN4R2 - WiFi capable device
* It has 320KB SRAM, 128KB ROM, 8MB PSRAM, and 4MB Flash memory
* USB-C male plug
* RGB addressable LEDs in eyes
* CircuitPython supported
* USB-HID capable

* Mouse jiggler
* "Bad" USB
* Ducky Script injector
